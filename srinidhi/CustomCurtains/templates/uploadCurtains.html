Upload Curtains Data From Excel


Zip File:
  1. It Should contain main folder with name curtains/shades with 'case sensitive'.
  2. default.jpg file('Image Not Available' Picture).
  3. skuids.zip file(this file should be compressed folder of any subfolder present in curtains/3D or shades/3D).
Main folder should contains:
  1. 3D
  2. 3D_second
  3. zoom
  4. for_zoom
  5. for_display
    and their respective sub folders.
Image Type:
  1. Swatches: If there is no availability of images of some skuids then select this option to use 'default.jpg' instead of 3D and 3D_second images.
  2. Render: If all the skuid images are available select this option.
Type:
  1. Curtains: Select this option while uploading curtains.
  2. Shades: Select this option while uploading shades.
Sub-Type: This Input will be shown when type Curtains is selected
  1. Normal: Select this option first to upload 3D, for_zoom and for_display folders images.
  2. Zoom: Select this option after uploading curtains normal to upload 3D_second & zoom folders images.
Shade/Curtain Types:
  1. Check the options of types which are present in the zip file(If all the option types are present in zip file, either u can check all or leave them uncheck).
Debug:
  1. True: This option is selected by default, select this option to test and find out the mistakes in all the files uploaded and options selected, whether are in right order.
  2. False: Select this option if there is no failed data in Debug True Mode and to upload the test/production.
